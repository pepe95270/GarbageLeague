# GarbageLeague

GarbageLeague est un petit jeu multijoueur au rythme déjanté où les joueurs doivent ramasser le plus de déchets et devenir le plus green des joueurs.

### Architecture

Ceci est un projet Unity en C#, il utilise l'architecture classique des projets pour Unity.
Chacun des dossiers parents corresponds à un type de fichiers utilisé par Unity (Materials, Scene, Sprites, etc)

Le dossier Script contient l'ensemble des scripts que nous avons développé. Puis chaque sous-dossier (dont les noms sont explicites) contient les scripts utiles à une partie bien spécifique de notre jeu (le joueur, les déchets, les zones, la gestion des inputs, etc).

### Ressources Open-Source utilisées:
* Standard Asset Unity: utilisé pour les contrôles de la voiture (en cours de remplacement par un algorithme fait maison)
* MKGlow: Shaders utilisés pour obtenir un effet "néon" sur certains de nos materials.
* Modèle 3D: nos modèles 3D viennent de différent modèle libre de droit trouvé sur le net.
